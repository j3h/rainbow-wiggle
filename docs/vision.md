# Vision

Last updated: 2026-03-02

## Product Vision
Create a super-fun web game that Lola (8) and Alder (10) can direct creatively and play on iPad Safari with zero install steps.

## Experience Pillars
- Instant joy: first interaction should trigger playful feedback.
- Coherent weirdness: bold kid ideas, but always one clear game loop.
- Tiny experiments: every change is reversible and test-backed.

## Success Criteria
- Fun in under 30 seconds: one tap should cause something delightful.
- Session-ready: new idea can be proposed, tested, and either kept or discarded in a single short session.
- Stable core: tests prevent regressions in interaction logic and iPad-web layout behavior.

## Scope Guardrails (Minimal-First)
- Start with one mechanic and one feedback loop.
- Each branch changes one thing only.
- If a feature cannot be demoed in under 2 minutes, split it into smaller steps.

## Current MVP
- Theme: Backyard disco animals.
- Main action: dance timing challenge.
- Judgment: 3 zones (Miss / Good / Perfect).
- Progress: rainbow meter rises with cool moves.
- Platform: iPad Safari-compatible web app.
