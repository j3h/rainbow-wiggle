# Creative Director Playbook

This is the session script for working with Lola and Alder.

## Session Rhythm (20-30 min)
1. Pick one tiny goal.
2. Build one reversible experiment.
3. Play-test immediately.
4. Decide: keep, tweak, or discard.

## Question Order
Ask these in order every session:

1. Warm-up (fun first)
- "What should make us laugh or cheer in the first 10 seconds?"
- Output: one delight moment.

2. Core action
- "What is the one button or move the player does the most?"
- Output: one primary action.

3. Feedback reward
- "When that action happens, what should we see/hear/change right away?"
- Output: one immediate feedback effect.

4. Win direction
- "How do we know we're doing well after one minute?"
- Output: one score/progress signal.

5. Constraint check
- "Should this be easy, medium, or chaotic?"
- Output: one difficulty target.

6. Tie-breaker
- "If we can only build one thing today, which one is most fun?"
- Output: single branch scope.

## Adult Translation (Engineering Manager)
- Convert outputs into a one-line branch goal.
- Reject any scope with more than one mechanic change.
- Ask for a go/no-go decision after play-test.

## Branch Card Template
- Branch: `codex/<topic>`
- Goal: `<single behavior change>`
- Test to add: `<what must remain true>`
- Demo step: `<how kids verify fun in <1 minute>`
- Decision: Keep / Tweak / Discard

## Stop Rules
- Stop when there are 2 failed experiments in a row; switch to play-testing existing build.
- Stop when a change cannot be explained in one sentence.
