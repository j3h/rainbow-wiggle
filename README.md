# Lola + Alder Game Lab

A web-first game playground where ideas can be tested in small, reversible steps.

## Goals
- Run in a browser on iPad (Safari) with no app install.
- Keep changes reversible through branch + commit workflow.
- Protect behavior with automated tests.
- Track product decisions and project vision in versioned docs.

## Quick Start
1. Run `npm test` to execute the automated test suite.
2. Run `npm run start` to serve the web app locally.
3. Open `http://localhost:4173` on desktop or iPad Safari.
